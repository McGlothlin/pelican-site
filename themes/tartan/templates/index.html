Welcome!

I am a software developer working for the CORE team at Oregon State University. Our team develops and maintains a reporting application that provides business intellignce for staff and faculty across campus.

I graduated from Oregon State University in 2015 with a Bachelors in Computer Science. Since then, I've had a taste of working all around the software stack. Some of my various projects have involved:

  • Data analysis and dashboard design
  • Front end web development
  • Back end web development
  • Linux and Solaris system administration

My favorite of these is front end web development. Recently at work I have taken on the role of a user interface (UI) designer for our system and I'm very excited about that; it's the type of work I've been looking for since I graduated. My responsibilities are split pretty evenly between report writing (lots of SQL) and UI design (JavaScript and CSS). When I'm not at work, I'm probably playing the bass guitar or learning about cars.
